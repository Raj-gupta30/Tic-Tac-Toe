# Tic Tac Toe Game

This is a simple Tic Tac Toe game built using Python.  
The project was created to practice Python basics such as functions, loops, conditionals, and lists.

## Features
- Two-player mode
- Input validation
- Win and draw detection
- Simple terminal-based interface

## Technologies Used
- Python

## What I Learned
While building this project, I learned:
- How to structure a Python program
- Working with functions and loops
- Handling user input
- Implementing game logic

## How to Run

1. Clone the repository
2. Open the project in VS Code
3. Run the following command in terminal:

```bash
python tic_tac_toe.py
